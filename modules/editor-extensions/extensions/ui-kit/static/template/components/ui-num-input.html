value

can drag to change value

step

If not set step, default step will be use by preference

<ui-num-input value="10" step="1"></ui-num-input>

preci

correct decimal places number
<ui-num-input value="10" preci="0"></ui-num-input> <ui-num-input value="10.123" preci="2"></ui-num-input> <ui-num-input preci="1" value="10.153"></ui-num-input>

Events

change // general case use this event confirm cancel unit-click

VUE JS

<ui-num-input unit="px" @change="onChange($event.target.value)" @confirm="onConfirm($event.target.value)" @cancel="onCancel($event.target.value)" @unit-click="onUnitClick" ></ui-num-input>
