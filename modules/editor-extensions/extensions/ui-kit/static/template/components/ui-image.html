channel

Show the channel data of one image.
The value is the combination of 'r','g','b','a'.
Usage cases: baked images in Light Map extension.
<ui-image value="{{assetUrl}}"></ui-image> <ui-image value="{{assetUrl}}" channel="r"></ui-image> <ui-image value="{{assetUrl}}" channel="g"></ui-image> <ui-image value="{{assetUrl}}" channel="b"></ui-image> <ui-image value="{{assetUrl}}" channel="a"></ui-image> <ui-image value="{{assetUrl}}" channel="rgb"></ui-image> <ui-image value="{{assetUrl}}" channel="rgba"></ui-image>
